# Data Portfolio – Especialización en Datos

Este repositorio contiene una colección de proyectos, prácticas y casos de estudio desarrollados durante mi proceso de formación en **Análisis de Datos, Ingeniería de Datos y Ciencia de Datos**.

Cada carpeta corresponde a un proyecto independiente e incluye:
- Código fuente
- Datos crudos (raw data)
- Documentación del caso (README propio)
- Resultados y conclusiones

El objetivo de este repositorio es demostrar habilidades prácticas en el manejo de datos, desde la exploración y análisis hasta la construcción de soluciones reproducibles y orientadas a negocio.
